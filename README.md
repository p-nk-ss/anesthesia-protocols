# Протоколи анестезіолога

Личный периоперационный справочник вспомогательных протоколов и алгоритмов по
анестезиологии — быстрый доступ к ключевым настановам с телефона прямо в
операционной. Весь контент и интерфейс — на украинском. Тёмная тема «монитор»,
работает офлайн (PWA).

> Личный справочник. Не заменяет клиническое суждение и действующие настановы.
> Дозы носят ориентировочный характер — всегда сверяйте с локальным протоколом,
> инструкцией производителя и состоянием пациента.

---

## Что внутри

Контент разбит по категориям; каждая страница — это либо пошаговый **алгоритм**
(вертикальная таймлиния с цветовой градацией важности), либо текстовая **справка**,
либо **интерактивный инструмент**.

### 🫁 Дыхательные пути
Сложная интубация трахеи у взрослых (по настанове DAS 2025).

### 🚨 Неотложные состояния
Самый большой раздел — покроковые когнитивные аиды периоперационных кризов:
анафилаксия, злокачественная гипертермия, LAST, бронхоспазм, кровотечение,
эмболия, ларингоспазм, аспирация, гиперкалиемия, реанимация новорождённого,
нарушения ритма (ACLS), гипо-/гипертензия, гипоксемия, капнография и др., плюс
принципы управления кризисом (CRM). Источники: **Stanford Emergency Manual v4.4**
и **ESA/EBA Emergency Quick Reference Guide** (с указанием атрибуции на каждой
странице).

### 🩺 Передоперационная оценка
Сопутствующие заболевания, текущая терапия, оценка дыхательных путей, индексы
риска и биомаркеры, рутинные предоперационные тесты (по ESA/NICE).

### 🧮 Шкалы и инструменты
Интерактивные калькуляторы (работают офлайн, считаются прямо в браузере):
- **Классификация ASA-PS**
- **STOP-BANG** — скрининг апноэ сна
- **AUDIT** — скрининг употребления алкоголя
- **PONV** — риск послеоперационной тошноты/рвоты (Apfel / POVOC)
- **Калькулятор инфузий** — вес → мл/час для непрерывных инфузий
- **Калькулятор доз препаратов** — вес → разовая (болюсная) доза в мг/мкг и объём
  в мл (индукция, опиоиды, миорелаксанты, реверсия, неотложные болюсы, антиэметики,
  максимальные дозы местных анестетиков), с ограничениями-капами

### ✅ Чек-листы
Интерактивный **хирургический чек-лист безопасности ВОЗ** (Sign In / Time Out /
Sign Out) с индикатором прогресса.

## Особенности

- **Офлайн-first (PWA)** — после первого открытия работает без сети, можно
  «установить» как приложение на телефон.
- **Тёмная тема «монитор»** — сигнатура: живой витальный след (ЭКГ / SpO₂ / EtCO₂).
- **Алгоритмы-таймлинии** — узлы подсвечиваются по мере прокрутки; важность шага
  кодируется цветом (info / success / warning / critical).
- **Интерактив без JS-зависимости** — калькуляторы и чек-листы построены на
  прогрессивном улучшении: базово работают даже без JavaScript.
- **Адаптив** — рассчитан на использование с телефона; крупные тач-цели, читаемость.
- **Поиск и фильтр по категориям** на главной.
- **Атрибуция источников** — каждая страница ссылается на свою настанову/источник.

Технологии: Astro (статическая генерация) + MDX, PWA (service worker),
самохостные шрифты (Spectral / Roboto / IBM Plex Mono). Без внешних трекеров и сети
в рантайме.
